# JobAt

## Usage

### Installation

#### 1. Clone Respository

```sh
git glone https://github.com/natebabyak/JobAt.git
```

#### 2. Install Dependencies

```sh
npm install
```

#### 3. Start Local Development Server

```sh
npm run dev
```
